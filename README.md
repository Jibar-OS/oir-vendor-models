# oir_vendor_models

Default model prebuilts for OpenIntelligenceRuntime (OIR).

- `models/qwen2.5-0.5b-instruct-q4_k_m.gguf` — Qwen 2.5 0.5B Instruct, Q4_K_M GGUF, Apache 2.0 (~491 MB)

Tracked via Git LFS. Install via `PRODUCT_PACKAGES += oir_default_model`
(from this repos Android.bp). Routes to `/product/etc/oir/` on the device.

Private until the OIR project public-release trigger fires.
